# MissClipper

Twitter のツイートを Misskey のクリップに保存するためのブラウザ拡張機能です。

画像もドライブに保存されます。動画は未対応です。

## インストール

### ビルド方法

1. 依存関係をインストールする

```bash
pnpm install
```

2. プロジェクトをビルドする

```bash
pnpm build
```

3. Chrome で拡張機能を読み込む
    - Chrome で `chrome://extensions/` を開く
    - 「デベロッパーモード」を有効にする
    - 「パッケージ化されていない拡張機能を読み込む」をクリック
    - ビルドされたディレクトリを選択

## 設定

拡張機能のオプションページで以下を設定してください。

- **Misskey ホスト**: 使用する Misskey インスタンスのURL
- **API トークン**: Misskey の API トークン
- **チャンネル ID** (オプション): ノートを投稿するチャンネルの ID
